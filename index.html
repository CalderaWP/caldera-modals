Simple Modals.

Create modals simply by adding inline attributes to your HTML

 <button type="button" 
    data-modal="mymodal"
    data-title="Example Modal"
    data-height="400"
    data-width="500"
    data-content="#modal-content">Click Me</button>

List of attributes available

Attribute Default Description
data-modal Required: id of the modal. activating a modal while a modal is already shown will reuse the ID or create a new one.
data-title Creates a title along with the closer button
data-height 550 sets the modal height
data-width 620 sets the modal width
data-padding 12 sets the padding of the modal content
data-title text string for title
data-content jQuery selector for element containing content or javascript callback function.
data-footer jQuery selector for element containing content, javascript callback function or text string
data-element div string indicating main modal wrapper type. eg. form or div etc.
data-animate 0 0 Requires a starting point value in X Y format e.g data-animate="0 -200px"
data-animate-speed 0.8s sets the speed of animation in s or ms. e.g 1s or 500ms
data-animate-ease ease sets the animation easing. [ ease | ease-in | ease-in-out | ease-in-out ]
data-autoload false sets the modal trigger to load on page load
data-sticky enables a sliding sticky modal and sets its position. e.g data-sticky="bottom right 50" sets the panel to the bottom right corner with a 50px offset.
data-focus false sets a modal with a close to be only closed with the closer click and not the backdrop. e.g data-focus="true"
data-speed sets the general speed of the fade and sliding actions (not animation)
